# 회고 (Retro)

작업 사이클을 마무리하고 배운 점을 정리하는 단계.

## 절차

1. `retrospector` 에이전트를 spawn하여 작업을 수행한다.
   - 작업 중 발견한 패턴, 반복된 실수, 개선점을 정리한다.
   - CLAUDE.md 업데이트 제안을 작성한다.
   - 기술 부채가 발견되었으면 기록한다.
2. `retro-reviewer` 에이전트를 spawn하여 산출물을 검증한다.
   - CLAUDE.md 업데이트 제안이 적절한가
   - 기존 규칙과 중복/충돌이 없는가
   - 실행 가능한 개선인가
3. 사용자 승인 후 CLAUDE.md를 업데이트한다.

## 산출물

- CLAUDE.md 업데이트 제안
- 기술 부채 기록 (해당 시)
